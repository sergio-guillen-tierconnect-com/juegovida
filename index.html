Tecnologías

El desarrollo fue realizado utilizando HTML5, utilizando las siguientes tecnologías:

  • KineticJS.- Framework para el trabajo con canvas, utilizado para la parte de gráficos, figuras y animación
  • Twitter Bootstrap.- Utilizado para la interfaz de usuario, estilos, etc.
  • jQuery.- Necesario para trabajar con Twitter Bootstrap y para controlar algunos eventos.

Herramientas

Se utilizo Sublime Text 2 como editor de texto, también las herramientas de depuración de Google Chrome

Esta aplicación tuvo como Sistema Operativo de prueba a GNU/Linux Debian.

Algunas jugadas

El juego de la vida tiene algunas jugadas peculiares y que se fueron estudiando a lo largo del tiempo.

Estas jugadas pueden Bodegones (Still lifes), Osciladores y Naves Espaciales.

  • Bodegones:
    • Block
    • Beehive
    • Loaf
    • Boat
  • Osciladores:
    • Blinker
    • Toad
    • Beacon
    • Pulsar
  • Naves espaciales:
    • Glider
    • Lightweight spaceship
